Talleres - Open Data Monterrey 2014

Sábado 22 de febrero

1:00 pm

Análisis espacial y estadístico de delitos: más allá de los hotspots y correlaciones

Carlos Castro - Estrategia Digital Nacional (México abierto)

Introducción al uso de datos, cartografía y visualizaciones sencillas utilizando programas de licencia libre como R y Octave. Grosso modo, se presentarán y explicarán técnicas estadísticas básicas de análisis de datos comparando su funcionamiento e interpretación de resultados.

En la segunda parte resolverá la siguiente pregunta: ¿cuál es el mejor lugar para colocar cámaras de seguridad en Monterrey para prevenir o reaccionar oportunamente ante la mayoría de los delitos? Utilizando la base pública de reportes del CIC, se propondrá una solución a este problema social utilizando las técnicas mostradas en la primera etapa.

3:00 pm

Civic hacking 101 - Creando tu primer aplicación cívica de código abierto

Braulio Chávez y Noé Domínguez - Codeando México

Introducción a datamx.io, la plataforma cívica de datos abiertos de México. Utilizando el API de datos de datamx.io se creará una visualización sencilla con D3.js, para después integrarla en una web app que subiremos a la nube.

Se presentará Git y GitHub para hacer tu aplicación open source y compartirla con otras personas.

5:00 pm

Scraping de datos públicos

Francisco Mekler - IMCO

Alguna vez has encontrado montones de datos útiles en websites, pero que no están para ser descargados? Tablas de información de sitios obsoletos del gobierno, párrafos con iniciativas, nombres y números valiosos, son datos que que podrías utilizar para construir una visualización, una infografía, o hasta una aplicación.

En este taller se presentará herramientas y técnicas en la práctica para scrapear esos datos de sitios web, como Python y la librería Pycurl.
